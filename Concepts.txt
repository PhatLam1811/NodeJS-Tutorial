Node JS là gì?
	- Môi trường thực thi (runtime environment)
	- Nền tảng hỗ trợ code JavaScript ở phía server

Engine V8
	- Engine open-source phát triển bời Google
	- Hỗ trợ dịch JavaScript về C++ và Assembly và ngược lại
	=> Giúp code JavaScript ở phía server

Học NodeJS là học gì?
	- Cách code serverside bằng JavaScript
	- Cách hoạt động của các công cụ sử dụng NodeJS
	- Không thực sự học về NodeJS

Front-end: phía client (browser)

Back-end: phía server (business logic)

Client-Server: Client request - Server response (HTML, CSS, JS...)

Web domain: tên miền (với user); địa chỉ IP (với máy tính)

Port của localhost: phân biệt web đang code với các chương trình khác đang cùng sử dụng localhost
